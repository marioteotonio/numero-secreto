# Número Secreto

## Descrição do Projeto

"Número Secreto" é um jogo interativo baseado em web desenvolvido com HTML, JavaScript e CSS. O objetivo do jogo é simples: o jogador deve adivinhar um número secreto gerado aleatoriamente pelo computador dentro de um intervalo especificado.

## Características Principais:
- **Interface Amigável**: Projetada com um layout simples e intuitivo para melhorar a experiência do usuário.
- **Lógica de Jogo Dinâmica**: O jogo fornece feedback instantâneo sobre as tentativas do jogador, indicando se o número é maior, menor ou se acertou, ajudando o jogador a melhorar suas estratégias.
- **Seleção de Intervalo**: Os jogadores podem escolher o intervalo para o número secreto, tornando o jogo acessível para diversos níveis de habilidade.
- **Rastreamento de Pontuação**: O jogo mantém o registro do número de tentativas, permitindo que os jogadores se desafiem e tentem bater seus próprios recordes.
- **Design Responsivo**: O jogo é otimizado para dispositivos desktop e móveis, garantindo que os jogadores possam desfrutar dele em qualquer lugar.

## Tecnologias Utilizadas:
- **HTML** para a estrutura do jogo.
- **CSS** para estilizar a interface do jogo.
- **JavaScript** para implementar a lógica do jogo e a interatividade.

## Instalação:
Para jogar, clone o repositório e abra o arquivo `index.html` em qualquer navegador web.

Aproveite o jogo e boa sorte adivinhando!
